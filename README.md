# PerforceTestingMCP

> A shared library of Claude Code **skills and commands** for getting more value out of
> Perforce's testing platforms via their MCP servers.

This repo packages reusable skills and commands that drive:

- **BlazeMeter MCP** — performance / load testing, including **Virtual Services** and
  **API Test & Monitoring**
- **Perfecto MCP** — mobile / web device testing and execution

Install the skills into Claude Code and use them to run, analyze, and automate testing
workflows against these platforms.

> 🚧 **Under active development.** Scope and structure are being defined — see the open
> issues and the PRD for the current plan.
